prompt和skill的区别？

- prompt
  - 反应式的，用户提问才有，被动触发
  - 一次性的，对话消失即结束
  - 不持久化的，无法跨会话复用
  - 它是越写越长，越勇越贵
- skill
  -  主动式的，AI自主决策时使用
  - 持久化的，安装后长期生效
  - 按需加载，用的时候才消耗token
  - 可组合使用的
  - 采用标准化封装，团队是可复用的
  - 本质上就是一个文件夹，存放
    - skill.md，第一部分是元数据，name和description是必填的，还有一些参数；再就是正文了，各种指令规则。
    - 运行的脚本
    - 参考资料
